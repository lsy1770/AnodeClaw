# 设备诊断指南

## 基本信息收集
1. 调用 `get_device_info` 获取设备基本信息
2. 调用 `get_battery_info` 获取电池状态
3. 调用 `get_memory_info` 获取内存使用情况

## 电池诊断
1. 检查电池电量和充电状态
2. 电量低于 20% 时建议充电
3. 检查是否有异常耗电应用

## 存储空间
1. 使用 `code_exec_async` 执行 `device.getStorageInfo()` 获取存储信息
2. 检查可用空间
3. 如果空间不足，建议清理大文件或缓存

## 性能检查
1. 获取内存信息，检查可用内存
2. 如果可用内存低于 500MB，建议关闭后台应用
3. 使用 `code_exec_async` 查看 CPU 负载

## 网络诊断
1. 使用 `http_request` 测试网络连通性（GET 一个可靠的 URL）
2. 检查 DNS 解析
3. 检查 WiFi/移动数据状态

## 常见问题处理
### 设备卡顿
1. 检查内存使用 → 关闭不必要的后台应用
2. 检查存储空间 → 清理缓存
3. 检查 CPU 温度 → 降温等待

### 应用崩溃
1. 查看应用日志
2. 尝试清除应用缓存
3. 检查应用版本是否最新

### 网络异常
1. 测试基本连通性
2. 检查 DNS 设置
3. 尝试切换 WiFi/移动数据
